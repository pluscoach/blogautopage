# BlogAutoFriends 랜딩페이지 설계 문서
## Claude Code 구현용 최종 확정본

---

## 1. 프로젝트 개요

### 무엇을 만드는가
네이버 블로그 서로이웃 자동 추가 AI 프로그램 "BlogAutoFriends"의 판매용 랜딩페이지.
결제(무통장 입금) → 텔레그램 알림 → 판매자 승인 → 이메일 자동 발송까지 연결되는 풀 자동화 시스템의 프론트엔드.

### 기술 요구사항
- **단일 HTML 파일** (가볍고 빠르게, 2초 이내 로딩 목표)
- **모바일 퍼스트** (네이버 블로그 유입 대부분이 모바일)
- **외부 프레임워크 최소화** (폰트, 아이콘 정도만 CDN)
- **Supabase 연동** (주문 폼 제출 시 orders 테이블에 INSERT)
- **반응형** (모바일 / 태블릿 / 데스크톱)

### 디자인 방향
- **컬러**: 네이버 초록(#03C75A) + 화이트(#FFFFFF) + 다크그레이(#1A1A1A) 기반
- **톤**: 밝고 신뢰감 있는 톤. AI가 만든 티 나지 않게. 쿠팡/네이버 느낌의 정보 밀집형이되 깔끔하게.
- **폰트**: Pretendard 또는 Noto Sans KR (한국 사용자 최적화)
- **핵심 원칙**:
  - F-패턴 기준 상단/좌측에 핵심 가치 배치
  - 점진적 공개(Progressive Disclosure)로 인지 부하 관리
  - 스티키 CTA 하단 고정 (모바일 필수)
  - 가격 앵커링 시각적으로 강조
  - AI 랜딩페이지 특유의 보라색 그라데이션, 뻔한 레이아웃 절대 금지

## 2. 섹션별 확정 카피 & 구현 가이드

### SECTION 1. 히어로 (첫 화면, 풀스크린)

**헤드라인:**
매일 3시간 '서이추·댓글 노가다'에 갇혀 계신가요?
이제 당신의 시간은 '콘텐츠'에만 쓰세요.

**서브카피:**
단순 반복은 AI가 맡습니다. 사람보다 더 사람처럼, 하루 100명과 쉼 없이 소통합니다.
컴퓨터만 켜두면 끝. 목표 수량을 채우면 AI가 스스로 작업을 종료하고 퇴근합니다.

**CTA 버튼 1:** 0원으로 24시간 무료 체험하기 (초록색 #03C75A, 크게)
**CTA 버튼 2:** AI의 소름 돋는 소통 방식 확인하기 ↓ (아웃라인 버튼, 스크롤 유도)

**우측(데스크톱) / 하단(모바일):** 프로그램 실행 화면 스크린샷 또는 GIF 영역 (placeholder 이미지)

**구현 노트:**
- 풀스크린 히어로, min-height: 100vh
- 헤드라인은 굵게, 서브카피는 가벼운 웨이트
- CTA 버튼 1에 미세한 pulse 애니메이션 (과하지 않게)
- 배경은 화이트 또는 매우 연한 초록 그라데이션

### SECTION 2. 문제 공감

**소제목:** 블로그 성장을 가로막는 '소통의 늪', 이런 적 없으신가요?

**카드 3개 (아이콘 + 텍스트):**

**카드 1 — 영혼 없는 품앗이, 남는 건 '숫자'뿐인 유령 이웃**
공들여 서이추를 보내도 돌아오는 반응은 전무합니다. 이웃 숫자는 늘어나는데, 내 글의 조회수와 체류 시간은 왜 제자리걸음일까요?

**카드 2 — 단 하루만 쉬어도 꺾여버리는 '활동 지수'의 공포**
지쳐서 이틀만 쉬면 반응이 뚝 떨어집니다. 블로그 성장을 위해 시작한 소통이 어느새 나를 옥죄는 '의무 노동'이 되어버렸습니다.

**카드 3 — '정성 댓글' 100명을 매일 손으로?**
의지만으로는 절대 지속할 수 없는 구조입니다. 하루 3시간의 단순 반복 작업, 애초에 사람이 계속하기에는 무리한 방식입니다.

**마무리 한 줄 (강조):**
당신의 잘못이 아닙니다. 지속 불가능한 '방식'이 문제였습니다.

**구현 노트:**
- 배경 연한 그레이(#F5F5F5) 또는 화이트
- 카드 3개 가로 배치(데스크톱) / 세로 스택(모바일)
- 각 카드에 심플한 아이콘 또는 이모지
- 마무리 문구는 볼드 + 초록색 하이라이트

### SECTION 3. 해결책 제시

**헤드라인:**
그래서, 블로거의 마음을 아는 **'AI 소통 전문가'**를 만들었습니다.

**본문:**
당신이 직접 소통할 때 거치는 모든 고민의 과정을
고성능 AI가 1:1 맞춤형으로 완벽하게 재현합니다.

컴퓨터만 켜놓으세요. AI가 타겟을 찾고, 글을 읽고, 100명과 소통한 뒤
당신의 블로그를 '살아있는 채널'로 만들고 조용히 종료합니다.

**구현 노트:**
- 섹션 전체를 다크 배경(#1A1A1A) + 화이트 텍스트로 전환 → 시선 리셋 효과
- 헤드라인 크게, 본문은 중앙 정렬
- 'AI 소통 전문가'에 초록색 강조

### SECTION 3.5. 사회적 증거 (신규 추가)

**소제목:** 이미 1,500명 이상의 블로거가 '노동'에서 해방되었습니다

**핵심 수치 4개 (인포그래픽 카드 가로 배치):**
- 누적 사용자: 1,582명 돌파
- 평균 이웃 수락률: 87.4% (단순 매크로 대비 4배 이상)
- 평균 방문자 수 증감: 사용 2주 후 평균 240% 상승
- 절감된 시간: 1인당 월평균 90시간 이상 (하루 3시간 기준)

**리얼 유저 후기 3개 (카드 뉴스 또는 메신저 캡처 스타일):**

**후기 1:**
"단순 숫자가 아니라, '진짜 팬'이 생기기 시작했어요"
"기존에 쓰던 프로그램은 이웃 수만 늘고 조회수는 그대로였는데, 이건 신기하게 답방이 정말 많이 옵니다. AI가 남긴 댓글에 상대방이 감동해서 대댓글을 달아주는 걸 보고 소름 돋았어요. 이제는 제가 손으로 댓글 안 답니다."
— 뷰티 인플루언서 루*님

**후기 2:**
"반신반의하며 시작했는데, 애드포스트 수익이 두 배로 늘었습니다"
"직장인이라 퇴근 후 3시간씩 소통하는 게 지옥 같았어요. 프로그램 돌려놓고 자고 일어나면 이웃 100명이 꽉 차 있습니다. 방문자 수가 안정적으로 유지되니까 블로그 지수가 올라가서 상단 노출도 전보다 훨씬 잘 되네요."
— 경제 블로거 재***님

**후기 3:**
"업체 블로그 운영하시는 분들, 무조건 쓰세요"
"로컬 사업자라 타겟팅이 중요한데, 저희 지역 카페나 경쟁사 이웃들만 콕 집어서 소통하니까 문의 전화 오는 횟수가 달라졌습니다. 사람 한 명 고용하는 것보다 100배는 경제적이에요."
— 스마트스토어 운영자 김*철님

**마무리 문구:**
단순한 '이웃 늘리기'가 아닙니다.
당신의 블로그를 '수익이 나는 공간'으로 바꾸는 데이터의 힘을 경험하세요.

**구현 노트:**
- 배경 화이트로 복귀
- 수치 4개는 큰 숫자 + 작은 설명 텍스트 조합 (카운트업 애니메이션 고려)
- 후기 카드는 메신저 말풍선 스타일 또는 인용 카드 스타일
- 각 후기에 살짝 다른 배경색 또는 좌측 초록 보더

### SECTION 4. 작동 방식

**소제목:** 사람이 하는 방식 그대로, AI의 정밀한 6단계 프로세스

**STEP 1 | 소통 타겟 정밀 분석**
관심 카테고리 내 게시글의 댓글창을 추적하여
단순 복붙이 아닌 정성 댓글을 남기는 '진짜 활동 유저'를 먼저 찾아냅니다.

**STEP 2 | 무의미한 리소스 낭비 차단**
이웃 수 초과, 서이추 차단 계정 등 신청해봤자 거절당할 계정은
AI가 미리 식별하여 리스트에서 즉시 제외합니다.

**STEP 3 | 전략적 서로이웃 신청**
내 이웃이 될 준비가 된, 수락 확률이 가장 높은
'알짜배기 계정'에만 정밀하게 서이추를 전송합니다.

**STEP 4 | 포스팅 맥락 정밀 분석**
단순히 글을 여는 것이 아닙니다. AI가 상대방의 최신 포스팅을
실제로 읽고 핵심 주제와 문맥을 완벽하게 파악합니다.

**STEP 5 | 소름 돋는 '맞춤형 정성 댓글' 작성**
"잘 보고 갑니다" 같은 뻔한 멘트가 아닙니다.
"내 글을 진짜 읽었네?"라는 감탄이 나오는 댓글로 강력한 첫인상을 남깁니다.

**STEP 6 | 안전 한도 달성 시 자동 종료**
네이버 가이드라인을 준수하는 하루 100명 한도 달성 시
별도 조작 없이도 프로그램이 스스로 안전하게 작업을 마칩니다.

**하단 한 줄 (강조):**
저희 프로그램은 결코 빠르지 않습니다. 실제 사람이 소통하는 '호흡'과 '속도'를 그대로 구현했기에 가장 안전하고 강력합니다.

**구현 노트:**
- 세로 타임라인/스텝퍼 UI (좌측 번호 + 우측 텍스트)
- 각 스텝에 간단한 아이콘
- 스크롤 시 각 스텝이 순차적으로 나타나는 애니메이션 (fade-in)
- 배경은 연한 그레이 또는 화이트

### SECTION 5. 핵심 차별점

**소제목:** 단순 매크로와 비교 불가, '압도적 수락률'의 비밀

**① 타겟 정밀 타격: 아무에게나 구걸하지 않습니다**
AI가 타겟 블로그의 소통 생태계를 분석하여 '소통에 진심인 사람'만 골라냅니다.
- 내 블로그와 결이 같은 카테고리별 맞춤 타겟팅
- 경쟁사 블로그에서 활발히 활동 중인 핵심 잠재 독자
- 특정 키워드를 선점한 실제 영향력 있는 블로거
- 직접 관리하는 아이디 리스트까지 완벽 대응
→ 서이추 수락률이 비약적으로 상승합니다.

**② 문맥 공감 댓글: '복붙'의 불쾌함을 지웠습니다**
상대방의 최신 글을 분석해 맥락에 맞는 댓글을 생성합니다.
"답방 오세요"라는 구걸성 멘트는 일절 사용하지 않습니다.
작성자가 답글을 달 수밖에 없는 지능형 공감 멘트를 생성합니다.
→ 첫 방문을 단골 팬으로 만드는 최고의 브랜딩입니다.

**③ 스마트 필터링: 헛수고 없는 효율적 관리**
- 이웃 풀(Full) 계정: 자동 스킵으로 슬롯 낭비 방지
- 광고/업체 블로그: 3단계 필터로 영혼 없는 계정 원천 차단
- 무반응 계정: 응답 없는 신청은 자동 취소하여 이웃 한도 최적화
→ 내 이웃 목록을 깨끗하고 활동적인 유저로만 채웁니다.

**구현 노트:**
- 3개 블록을 카드 또는 아코디언으로 구성
- 각 차별점에 아이콘 (타겟/댓글/필터)
- 화살표(→) 뒤의 결론 문구는 초록색 볼드

### SECTION 6. 경쟁 비교

**소제목:** 노동 vs 매크로 vs 본 프로그램, 한눈에 비교하세요

| 구분 | 손으로 직접 하는 소통 | 일반 저가형 매크로 | 본 AI 프로그램 |
|------|---------------------|------------------|--------------|
| 내 시간 소모 | 하루 3시간 이상 (고통) | 매번 복잡한 설정 필요 | 클릭 한 번이면 끝 |
| 타겟 선별 | 주관적 판단 (느림) | 무작위 스팸 발송 | AI 기반 '진짜 유저' 분석 |
| 댓글 품질 | 정성껏 직접 작성 | "잘 보고 갑니다" 무한 복붙 | 포스팅 맞춤형 지능형 댓글 |
| 필터링 | 일일이 눈으로 확인 | 필터링 기능 없음 | 이웃 초과·광고 계정 자동 스킵 |
| 계정 안전성 | 안전하지만 체력의 한계 | 스팸 신고 및 블록 위험 | 인간 패턴 구현, 압도적 안전 |
| 지속 가능성 | 며칠 못 가 포기 | 저품질 위험 누적 | 매일 100명씩 영구적 성장 |
| 소통 효과 | 높지만 지속 불가능 | 매우 낮음 (차단 급증) | 매우 높음 & 자동화 성공 |

**구현 노트:**
- 반응형 테이블 (모바일에서는 카드형으로 전환)
- "본 AI 프로그램" 열은 초록 배경 하이라이트
- 다른 두 열은 회색 톤

### SECTION 7. 안전성 안내

**소제목:** "내 블로그는 소중하니까", 계정 안전을 최우선으로 설계

단순한 대량 작업은 블로그를 망치는 지름길입니다. 저희는 철저히 '네이버 친화적'으로 움직입니다.

- **인간 패턴 완벽 구현:** 실제 사람이 타이핑하고 스크롤하는 속도를 그대로 시뮬레이션합니다.
- **변동 확률 딜레이:** 기계적인 반복이 아닌, 매번 다른 랜덤 딜레이로 네이버가 봐도 자연스러운 소통 패턴을 유지합니다.
- **클린 타겟팅:** 활동 유저만 골라 소통하므로 스팸 신고 확률을 최소화했습니다.
- **100% 안심 환불 보장:** 현재까지 본 프로그램 사용으로 인한 제재 사례는 단 1건도 없습니다. 만일 불이익 발생 시, 남은 기간을 이유 불문 전액 환불해 드립니다.

**구현 노트:**
- 배경 연한 초록(#E8F5E9) 또는 화이트
- 방패/체크 아이콘 활용

### SECTION 10. FAQ

**Q1. 블로그에 불이익이 생기지 않을까요?**
네이버가 권장하는 '정성 소통'의 로직을 AI가 그대로 따릅니다. 무차별적인 매크로와 달리 랜덤 딜레이와 행동 패턴 분석이 적용되어 매우 안전합니다.

**Q2. 기계치인데 세팅이 어렵진 않나요?**
설치 후 타겟 카테고리만 클릭하면 세팅이 끝납니다. 상세 가이드와 영상은 물론, 막히는 부분은 1:1 채팅으로 실시간 도와드립니다.

**Q3. 여러 개의 블로그 계정을 돌려도 되나요?**
네, 계정 수 제한은 없습니다. 다만 안전을 위해 한 PC에서 여러 계정 작업 시 순차 실행을 권장하며 관리 노하우도 함께 전달드립니다.

**Q4. 여러 대의 PC에서 쓸 수 있나요?**
인증 코드 1개당 1개의 PC에서만 사용 가능합니다. PC 변경이 필요하시면 채팅으로 문의해 주세요.

**Q5. 맥(Mac) 환경에서도 돌아가나요?**
현재 윈도우(Windows) 전용으로 개발되었습니다. 맥 유저분들은 부트캠프나 가상 윈도우 환경에서 이용 가능합니다.

**Q6. 체험 후 결제하면 어떤 혜택이 있나요?**
무료 체험 후 정식 구매 시, 첫 달 이용 기간 추가 등 특별 시크릿 혜택을 드립니다. 구매 후 채팅 상담 시 말씀해 주세요.

**구현 노트:**
- 아코디언 UI (클릭하면 답변 펼쳐지는 방식)
- Q는 볼드, A는 일반 웨이트
- 열린 상태에서 좌측에 초록 보더

## 3. Supabase 연동 스키마

### orders 테이블
```sql
CREATE TABLE orders (
  id BIGSERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT NOT NULL,
  plan TEXT NOT NULL,          -- 'free_trial' | 'monthly' | 'full_package'
  amount INTEGER DEFAULT 0,   -- 0 | 39000 | 59000
  status TEXT DEFAULT '입금대기', -- '입금대기' | '결제완료' | '발송완료'
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

```sql
CREATE TABLE licenses (
  id BIGSERIAL PRIMARY KEY,

  buyer_name TEXT NOT NULL,
  is_active BOOLEAN DEFAULT TRUE,
  expires_at TIMESTAMPTZ,
  last_active_at TIMESTAMPTZ,
  last_machine_id TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

```
[고객] 폼 제출

    → 이메일 자동 발송
      - 프로그램 다운로드 링크 (구글 드라이브)
      - 라이선스 코드
      - 설치 가이드
    → Supabase orders status → '발송완료'
```
