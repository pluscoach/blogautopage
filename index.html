Q2. 하루에 시간이 얼마나 걸리나요? ▼
PC 켜두기만 하면 됩니다. 최초 설정은 가이드 영상 보면서 5분이면 끝나고, 이후부터는 클릭 한 번이면 자동으로 실행됩니다.

하루 1~2시간 동안 알아서 100명과 소통합니다. 실행 중에 다른 작업을 하셔도 되고, 외출하셔도 됩니다. 완료되면 자동으로 종료됩니다.
Q3. API 비용이 별도로 드나요? ▼
네, AI 기능에 Gemini API를 사용하며, API 비용은 솔루션 가격에 포함되어 있지 않습니다. 사용량에 따라 별도로 발생합니다.

이웃 관리 기준 하루 100명 소통 시 약 30~50원 수준이며, 글쓰기 에디터는 글 1편당 약 50~60원 수준입니다. 이미지 생성 사용에 따라 달라질 수 있습니다.

가이드 영상에서 API 키 발급 방법과 비용 관리법을 안내해드리고 있습니다.
Q4. 맥(Mac) 환경에서도 돌아가나요? ▼
현재 윈도우(Windows) 전용으로 개발되었습니다. Mac 버전은 추후 개발 예정이며, 현재는 준비 중에 있습니다.

다만, Mac 유저분들도 Parallels, VMware, 부트캠프 등 가상 윈도우 환경을 통해 실행하실 수 있습니다. 실제로 가상 환경에서 사용하시는 분들도 계시며, 정상적으로 작동합니다.

가상 환경 설치가 어려우신 분은 카카오톡으로 문의해주시면 안내해드리겠습니다.
Q5. 실제로 효과가 있나요? ▼
실제로 사용 2주 후 방문자 24명 → 980명(41배 증가) 사례가 있습니다.

이웃 100명이 매일 쌓이면서 블로그 지수가 올라가고, 검색 노출이 자연스럽게 늘어납니다. 단순히 숫자만 채우는 게 아니라 실제 소통하는 이웃이기 때문에 재방문율도 높습니다.

다만, 무조건 방문자가 느는 것은 아닙니다. 현재 운영 중인 블로그에 게시글이 충분하지 않거나, 글의 퀄리티가 낮으면 이웃이 늘어도 방문자 수에 큰 변화가 없을 수 있습니다.

결과는 블로그 계정의 운영 상태, 게시글 수, 글의 주제와 퀄리티에 따라 달라집니다. 솔루션은 이웃 관리를 자동화해주는 도구이며, 100% 결과를 보장하는 것은 아닙니다. 꾸준히 글을 쓰면서 함께 사용하실 때 가장 좋은 효과를 보실 수 있습니다.
