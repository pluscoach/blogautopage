## 4. 1차 광고에서 얻은 인사이트

### 4-1. 잘 된 것

✅ **CAPI 시스템 정상 작동**
- 실제 구매 1건이 Server 출처로 Meta에 정확히 인식
- 시스템 인프라는 100% 검증 완료

✅ **Instagram 비중 80%**
- B2C/마케팅 타겟층은 Instagram이 압도적
- Facebook 페이지만 했으면 광고비 80% 손실

✅ **성별 균형**
- 편향 없이 광고 노출 → 타겟층이 양성에 고르게 분포

✅ **25-44세 타겟 적중**
- Meta AI가 의도한 연령대를 자동으로 잘 찾음

### 4-2. 개선 필요한 것

🟡 **카피의 모호성**
- "상위 1%", "비밀 무기" → 추상적
- 구체적 페인 포인트 부재
- 누구의 어떤 문제를 해결하는지 불명확

🟡 **타겟 페르소나 부재**
- "블로그 운영하는 사람" 너무 광범위
- 진짜 구매 의향 있는 사람 = 시간 부족하고 글 많이 써야 하는 사람

🟡 **데이터 부족으로 깊은 분석 불가**
- 구매 1건은 통계적으로 의미 X
- 5건 이상 쌓여야 진짜 분석 가능

### 4-3. 새로운 타겟 정의 (사장님 결정)

**핵심 타겟: 블로그 마케팅 대행사 / 마케터**

세부 페르소나:
- **블로그 마케팅 대행사 사장/직원** (B2B → C2C 전환점)
- **자영업자**: 블로그 마케팅 직접 하는 분
- **체험단/원고료 수익 블로거**: 매일 글 2~3개 써야 하는 사람
- **클라이언트 다수 관리하는 프리랜서 마케터**

**이들의 진짜 페인:**
1. 글 1개 쓰는 데 3시간 → 시간 부족
2. 클라이언트 10곳 글 혼자 감당 불가
3. AI 써도 네이버에서 안 잡힘
4. 매일 글 못 올려서 수익 정체

### 6-1. Phase 1: 사이트 카피 재정비 (1~3일)

**"누구한테 말하는 광고인지"** 사이트 첫 화면에서 즉시 명확하게.

#### 우선 손볼 섹션

**A. 메인 헤드라인 (Hero 섹션)**
- 현재: "상위 1% 블로거들의 비밀 무기 / AI 소통 관리 솔루션"
- 변경 방향:
  - 타겟 명시: "블로그 마케팅 대행사 / 마케터"
  - 페인 포인트 직접 언급
  - 예시 후보:
    - **"매일 클라이언트 글 10개, 손으로 다 쓰세요?"**
    - **"블로그 대행 마케터, AI로 1/30 시간에 끝내세요"**
    - **"체험단 블로거 - 하루 3건 글쓰기, 이제 2분이면 됩니다"**

**B. "이런 분들께 추천" 섹션 신설**
- 4~5가지 페르소나로 명확히 분류
- 각 페르소나별 구체적 페인 포인트 명시
- v3 가이드의 표 형식 참고

**C. 후기/사례 섹션 보강**
- 단순 텍스트 후기 → 실제 사용 결과 캡처
- "Before: 하루 3시간" → "After: 6분"
- 가능하면 구체적 수치 포함

**D. CTA 문구 재검토**
- "지금 바로 시작하기" → 타겟 맞춰 변경 가능
- 예: "마케팅 대행사 사장님 보세요" 같은 페인 직접 자극

#### 안 건드릴 곳
- 가격 (이미 변경 완료)
- 결제 시스템
- AI 블로그 에디터 섹션 (괜찮음)
- 선착순 이벤트 박스

### 6-2. Phase 2: 광고 이미지 재제작 (1일)

#### 기존 이미지 문제
- "절대 손으로 / AI로" → 시각적으로는 좋으나 메시지 불명확
- 누구를 위한 광고인지 첫눈에 안 보임

#### 새 이미지 방향

**옵션 A: 페르소나 + 페인 직접 언급**
- 텍스트 예: "블로그 마케팅 대행사 사장님!"

- 시각적: 시계/스트레스 받는 사람 → AI로 변환
