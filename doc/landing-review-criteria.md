## A. 구조/흐름 기준

### A-1. 피크 직후 CTA 배치 (최우선)

- [ ] 수익 데이터 섹션(구 슬라이스 8, 훅 8.5) 직후에 CTA 버튼 존재
- [ ] AI 기능 소개 섹션(구 슬라이스 3, 훅 8.2) 근처에도 CTA 존재
- [ ] CTA가 페이지 말미(구 슬라이스 16)에만 몰려있지 않음

### A-2. 중간 사망 구간 제거

- [ ] 구 슬라이스 5~6(필요성 3.9~4.6) 해당 콘텐츠가 삭제 또는 재배치됨
- [ ] 피크(AI 기능 소개) → 수익 증거 사이에 흐름을 끊는 불필요한 섹션 없음
- [ ] 피크 직후 참여도가 급락하는 구간이 없음 (브리지 콘텐츠 삽입 여부)

### A-3. 말미 압축

- [ ] 슬라이스 15~16 해당 구간이 대폭 축소됨
- [ ] 구매 결심 포인트와 CTA 사이 거리가 8섹션 이하로 줄어듦
- [ ] 이미 결정한 유저가 바로 행동할 수 있는 경로 존재

### A-4. 전체 길이

- [ ] 페이지 전체가 이전(16슬라이스)보다 짧거나 밀도가 높아짐
- [ ] 각 섹션이 하나의 명확한 역할을 가짐 (중복 메시지 없음)

## B. 메시지 전달 기준

### B-1. 핵심 차별화 (현재 전달률 10~15% → 목표 50%+)

- [ ] "유령 이웃이 아닌 진짜 소통하는 이웃 발굴" 메시지가 상단(히어로~첫 3섹션)에 명시됨
- [ ] 이웃 수 증가 → 실제 비즈니스 전환(매출/문의/계약) 연결이 시각적으로 보임
- [ ] "단순 숫자 늘리기 도구"가 아닌 "질적 소통 솔루션"으로 포지셔닝됨

### B-2. 신뢰 구축 (현재 5.25/10 → 목표 7.0+)

- [ ] 업종별 구체적 성공 사례/글 샘플 존재 (14/20명 요구)
- [ ] 실제 블로그 URL 또는 네이버 상위 노출 스크린샷 존재 (10/20명 요구)
- [ ] 3개월 이상 장기 사용자 후기 존재 (6/20명 요구)
- [ ] 이웃 수 → 실제 매출/전환 데이터 존재 (9/20명 요구)

### B-3. 불안 해소

- [ ] 네이버 계정 제재/저품질 리스크 해소 내용 명시 (12/20명 우려)
- [ ] AI 감지 우회 실제 증거(스크린샷 등) 존재 (5/20명 요구)
- [ ] "안전성 보장" 콘텐츠가 구매 결심 구간 근처에 배치됨 (슬라이스 7~9 사이)

### B-4. 표현 톤

- [ ] "노가다"라는 표현 제거 또는 대체 ("루틴 자동화", "시간 해방" 등)
- [ ] 과장 마케팅 문구 최소화 (6/20명 피로감 표현)
- [ ] 가격 인상 압박 톤을 "얼리어답터 혜택" 프레이밍으로 전환 (4/20명 반감)

## C. 전환 퍼널 기준

### C-1. 무료체험 동선 (전달률 75%, 가장 강력한 전환 레버)

- [ ] 무료체험 CTA가 페이지 상단~중단에 최소 1회 노출
- [ ] "내 키워드로 즉시 글 생성 체험" 경험이 명확히 안내됨 (16/20명 결제 조건)
- [ ] 무료체험 → 결제 전환 경로가 자연스러움

### C-2. 가격 제시

- [ ] 가격이 충분한 가치 입증 후에 등장
- [ ] "대행사 월 50만원 vs 직접 관리 월 5.9만원" 비교가 명확
- [ ] 가격 카드가 혼란 없이 깔끔 (무료체험 / 1개월 / 2개월)

## D. 타겟 적합성 기준

### D-1. 1순위 타겟 (구매 의향 높은 그룹)

| 그룹 | 구매 의향 | 핵심 트리거 |
|------|----------|------------|
| 영업직 종사자 | 7.25 | ROI 계산 ("보험 한 건 > 월 5.9만원") |
| 인플루언서/체험단 | 6.5 | AI 감지 우회 + 시간 절약 |
| 자영업자/소상공인 | 5.25 | 대행비 비교 + 직접 컨트롤 |

- [ ] 카피가 위 3그룹의 언어/상황에 맞춰져 있음
- [ ] 각 그룹이 "나를 위한 서비스"라고 느낄 수 있는 사례/문구 존재

### D-2. 제외/주의 타겟

| 그룹 | 구매 의향 | 문제 |
|------|----------|------|
| 1인 사업자(전문직) | 4.0 | "AI 글이 전문성 훼손" 인식, 공유 0% |
| 수익형 블로거(베테랑) | 5.25 | 기존 AI 도구 피로감 |

- [ ] 전문직/컨설턴트를 직접 타겟하는 문구 없음 (역효과 방지)
- [ ] 베테랑 블로거의 "또 하나 나왔네" 피로감을 유발하는 과장 표현 없음
