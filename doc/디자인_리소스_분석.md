## 2. CSS 애니메이션 (animate.css)

### 우리 페이지 섹션별 추천 애니메이션

| 섹션 | 추천 애니메이션 | 효과 |
|------|---------------|------|
| 히어로 텍스트 | `fadeInUp` | 아래→위 자연스러운 등장 |
| CTA 버튼 | `pulse` (무한반복) | 주목도 향상 |
| 말풍선 (고민) | `fadeInLeft` | 채팅처럼 왼쪽에서 등장 |
| 가격 카드 | `zoomIn` | 스크롤 시 확대 등장 |
| 후기 카드 | `fadeIn` | 부드러운 등장 |
| FAQ 답변 | `fadeInDown` | 아코디언 열림 효과 |
| 숫자 (41배, 3배) | `bounceIn` | 임팩트 있는 등장 |

### 적용 방법
```html
<!-- CDN 추가 -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css">

<!-- 사용 예시 -->
<h1 class="animate__animated animate__fadeInUp">블로그 관리,</h1>
```

## 3. AOS (Animate on Scroll) — 가장 추천

스크롤 시 요소가 나타나는 효과. **우리 페이지에 가장 즉시 적용 가능.**

### 적용 방법
```html
<!-- CDN -->
<link href="https://unpkg.com/aos@2.3.1/dist/aos.css" rel="stylesheet">
<script src="https://unpkg.com/aos@2.3.1/dist/aos.js"></script>
<script>AOS.init({ duration: 800, once: true });</script>

<!-- 사용 -->
<div data-aos="fade-up">이 요소는 스크롤하면 아래→위로 등장</div>
<div data-aos="fade-up" data-aos-delay="200">0.2초 뒤 등장</div>
```

### 섹션별 적용 계획

| 섹션 | AOS 효과 | delay |
|------|---------|-------|
| 히어로 타이틀 | `fade-up` | 0 |
| 말풍선 1 | `fade-up` | 100 |
| 말풍선 2 | `fade-up` | 200 |
| 말풍선 3 | `fade-up` | 300 |
| "0분으로" | `zoom-in` | 0 |
| AI 데모 카드 | `fade-up` | 0 |
| 결과 증거 (41배) | `zoom-in` | 0 |
| STEP 1~4 카드 | `fade-right` | 각 100씩 |
| 가격 카드 | `zoom-in` | 각 100씩 |
| 후기 이미지 | `fade-up` | 각 100씩 |

## 즉시 적용 우선순위

### 1순위 (5분, CDN만 추가)
- **AOS** 스크롤 애니메이션 → 전체 섹션에 `data-aos="fade-up"` 적용

### 2순위 (30분)
- **SVG Wave/Shape Divider** → 점 3개 대체, 섹션 간 자연스러운 전환
- **animate.css** → CTA 버튼 pulse, 말풍선 fadeInLeft

### 3순위 (1시간)
- **Storyset/unDraw 일러스트** 다운 → 이모지 대체
- **LottieFiles** 애니메이션 1~2개 → 히어로 또는 "0분으로" 섹션

### 4순위 (개선)
- **Heroicons** SVG 아이콘 → STEP 1~4 아이콘 교체
- **숫자 카운터 애니메이션** → 41배, 3배 등 스크롤 시 카운트업
- **색상 대비 검수** → whocanuse.com에서 테스트
