## 5. 기능·베네핏 매핑

### 이웃 관리 솔루션

| 기능 | 베네핏 |
|------|--------|
| STEP 1: 타겟 분석 (소통이웃 자동 선별) | 시간: 수동 검색 불필요 / 감정: 스팸 이웃에게 시간 낭비 안 함 |
| STEP 2: 이웃 신청 (수락률 높은 계정에 선별 발송) | 돈: 거절당할 계정에 리소스 낭비 차단 / 시간: 50명 자동 발송 |
| STEP 3: 정성 댓글 (포스팅 맥락 분석 후 댓글) | 감정: "내 글을 진짜 읽었네?" 감탄 유발 → 답방 확률 상승 |
| STEP 4: 이웃 관리 (스팸·무반응 자동 정리) | 시간: 수동 정리 불필요 / 감정: 깔끔한 이웃 목록 유지 |
| 인간 패턴 정밀 구현 | 감정: 계정 안전, 불이익 걱정 해소 |
| 변동 확률 딜레이 (3~45초) | 감정: 자연스러움, 매크로 탐지 회피 |
| 클린 타겟팅 | 감정: 스팸 신고 확률 최소화, 안심 |

### AI 글쓰기 에디터

| 기능 | 베네핏 |
|------|--------|

| 원클릭 복사 → 블로그 에디터 붙여넣기 | 시간: 발행 과정 단축 |

## 6. 톤 & 보이스 가이드

### 어조
- **존댓말** 기반 (합쇼체 + 해요체 혼용)
- **도발적 질문**으로 시작 → **안도감** 제공 → **행동 유도**
- 과장 없이 **구체적 숫자**로 설득

### 자주 쓰는 어미
- "~하세요" (고용하세요, 맡기세요, 확인해보세요)
- "~입니다" (끝입니다, 충분합니다, 다릅니다)
- "~합니다" (출근합니다, 퇴근합니다, 관리합니다)

### 핵심 비유·프레이밍
- **"직원 고용"** — 프로그램 구매가 아닌 사람 고용 프레임
- **"팀장급 직원"** — 일반 직원이 아닌 책임감 있는 인재
- **"출근 / 퇴근"** — AI를 사람처럼 의인화
- **"월 120만원 vs 7만원대"** — 비용 앵커링
- **"버튼 한 번"** — 극도의 간편함

## 7. 신뢰 요소 (Social Proof / Risk Reversal)

### Social Proof
- **실제 사용자 후기** — 이미지 6장 (2열 그리드, 클릭 확대)
- **업무 보고서 UI** — "이웃 신청 100명 완료 ✓ / 정성 댓글 100개 작성 ✓ / 스팸 31명 차단 ✓ / 1시간 47분"
- **UPDATE LOG** — v3.4.0 (2026.04.23), v3.2.0 (2026.01.03), v3.1.2 (2025.12.23) — 지속적 업데이트 증거

### Risk Reversal
- **안심 환불 정책** — "솔루션 사용으로 인한 불이익 발생 시, 남은 기간에 대해 환불"
- **무료 체험** — "카카오톡 채널로 신청 시 1일 사용권 제공"
- **24h 고객 지원**
- **가격 영구 동결** — "지금 구매하시면, 앞으로 가격이 올라도 현재 가격 그대로"

### 긴박감 (Scarcity / Urgency)
- **15자리 제한** — "매달 15명만 관리합니다. 더 받지 않습니다."
- **실시간 공석 카운터** — "현재 공석 3 / 15자리" (DB 조회)
- **가격 인상 이력** — 1차 39,000 → 2차 59,000 → 3차 73,000 (현재) → 4차 99,000~ (예정)
- **4차 업데이트 공지** — "블루오션 키워드 자동 발굴 → 현재 73,000원 → 99,000원~"
