## 1. 섹션별 페르소나 분석

### SECTION 1: HERO

**내용:** "유령 이웃 대신 잠재 고객으로 바꿔드립니다" + 타임라인 데모 카드 + "7일 무료 체험" CTA

| 페르소나 | 반응 | 스크롤 지속 |
|---------|------|-----------|
| 김예빈 (인플루언서) | "유령 이웃" 공감. "하루 100명" 매력적. 다만 대행 계정 여러 개 관리 가능 여부 의문 | 85% |
| 윤윤숙 (영업직) | "잠재 고객 확보"에 반응. 데모카드 UI 다소 복잡 | 80% |
| 김동혁 (자영업자) | "유령 이웃" 모를 수 있음. "이게 내 매출에 어떻게?" 불분명 | 65% |

**문제:**
- "블로그 지수를 높이는 최고의 솔루션" 서브카피 — 블로그 운영 경험자만 이해
- CTA가 #solution-section(HOW IT WORKS)으로 가는데, 사용자는 무료 체험 폼을 기대

### SECTION 2: HOW IT WORKS

**내용:** "PC만 켜두면 자동으로, 하루 100명" → 4스텝 → 영상 → 41배 증가 + CTA

| 페르소나 | 반응 | 스크롤 지속 |
|---------|------|-----------|
| 김예빈 | 4스텝 구체적. 92% 필터링 UI 인상적. 41배 "특수 사례?" 의문 | 90% |
| 윤윤숙 | "PC만 켜두면" 핵심 안심. 댓글 예시가 자연스러움 | 85% |
| 김동혁 | "하루 100명" 와닿지만 ROI 연결 부재. 업종별 사례 없음 | 60% |

**문제:**
- 41배 증가 데이터에 업종/맥락 정보 없음 → "특수 사례" 의심
- 실제 블로그 URL, 네이버 상위 노출 스크린샷 없음 (기준서 10/20명 요구)

### SECTION 3: 실제 사용자 후기

**내용:** 카카오톡 대화 캡처 이미지 6~7개

| 페르소나 | 반응 | 스크롤 지속 |
|---------|------|-----------|
| 김예빈 | 카톡 형태 익숙. 대행 관련 사례 확인 필요 | 80% |
| 윤윤숙 | 이미지 작게 보임 (55세 시력). 텍스트 후기 없음 | 75% |
| 김동혁 | "카톡 캡처는 조작 가능" 의심. **첫 번째 주요 이탈 포인트** | 55% |

**문제:**
- 이미지만 구성 — 텍스트 스캔 불가
- 업종별 사례, 장기 사용자 후기 메타데이터 없음

### SECTION 4: PRICING ⚠️

**내용:** 무료체험(0원/5일), 1개월(79,000원), 2개월(129,000원)

| 페르소나 | 반응 | 스크롤 지속 |
|---------|------|-----------|
| 김예빈 | 가격 저렴. 근데 "글쓰기 3개월 체험권"이 여기서 나오는데 아직 에디터 소개 안 봄 | 70% |
| 윤윤숙 | 3단 비교 이해 쉬움. 디지털 약자 지원(1:1 세팅) 안내 없음 | 65% |
| 김동혁 | 79,000원이 비싼지 싼지 기준 없음 (대행사 비교 없음). **두 번째 주요 이탈 포인트** | 50% |

**문제 (심각):**
- ❌ **PRICING이 AI 에디터 섹션보다 먼저 나옴** → "글쓰기 3개월 체험권"을 아직 모르는 상태에서 가격 노출
- ❌ 가치 입증 부족 상태에서 가격 → C-2 FAIL
- ❌ 대행사 비교("월 50만원 vs 월 6.5만원") 없음
- ❌ lifetime 플랜이 form에만 있고 가격 카드에 없음

### SECTION 5: AI 블로그 에디터 (다크 섹션) ⚠️

**내용:** 브릿지 → 에디터 데모 → AI 감지 우려 → 판독기 통과 증거 → 3개월 체험권

| 페르소나 | 전환 기여 | 문제 |
|---------|---------|------|
| 김예빈 | 높음 (AI 감지 우회가 핵심 트리거) | pricing 뒤에 있어 이탈 후 도달 못할 수 있음 |
| 윤윤숙 | 중간 (다크 UI + 작은 텍스트 = 읽기 어려움) | 섹션 전체가 너무 김 (670줄, 전체의 40%) |
| 김동혁 | 도달 시 높음 | pricing에서 이미 이탈했을 가능성 높음 |

**문제 (심각):**
- ❌ **가장 설득력 있는 콘텐츠가 pricing 뒤에 숨겨져 있음** → 대부분 도달 못함
- ❌ 섹션 670줄 = 전체의 40%. "..." 패딩 과다, 화살표 3세트 반복
- ❌ 쿠폰 "정가 87,000원"의 근거 불명 (pricing에 해당 가격 없음)

## 5. 무료 체험 전환 플로우 평가

**현재 문제:**
- ❌ **히어로 "7일 무료 체험" vs pricing "5일 무료 체험"** — 기간 불일치 (신뢰 문제)
- ❌ 히어로 CTA가 pricing이 아닌 설명 섹션으로 이동 — 기대 불일치
- ❌ 무료체험이 카카오톡 수동 안내 → 즉시 체험 불가 → 전환율 하락
- ❌ "구매 후 체험권 받기" 버튼이 scrollToForm('free') 호출 — 무료인지 유료인지 혼란

## 6. Criteria Pass/Fail

| 항목 | 판정 |
|------|------|
| A-1 피크 직후 CTA | **FAIL** — AI 기능이 pricing 뒤 |
| A-2 중간 사망 구간 | **FAIL** — 후기→pricing 가치 미입증 |
| A-3 말미 압축 | **PARTIAL** |
| A-4 전체 길이 | **FAIL** — 다크 섹션 비대 |
| B-1 핵심 차별화 | **PASS** |
| B-2 신뢰 구축 | **FAIL** — 업종별 사례 0, URL 0, 매출 데이터 0 |
| B-3 불안 해소 | **PARTIAL** — FAQ에만 있고 pricing 근처에 없음 |
| B-4 표현 톤 | **PARTIAL** — 카운트다운 압박톤 유지 |
| C-1 무료체험 동선 | **FAIL** — 카카오톡 수동 + 기간 불일치 |
| C-2 가격 제시 | **FAIL** — 대행사 비교 없음, 가치 입증 전 노출 |
| D-1 타겟 적합성 | **PARTIAL** |
| E 김예빈 | **PARTIAL** |
| E 윤윤숙 | **FAIL** — 디지털 약자 지원 없음 |
| E 김동혁 | **FAIL** — ROI 불분명, 대행사 비교 없음 |

## 7. 구조 개선 권고 (우선순위)

### P0 (즉시 수정)
1. **무료체험 기간 통일:** 히어로 "7일" → "5일"로 통일
2. **중복 ID 제거:** solution-section ID 중복
3. **판독기 슬라이더 JS 버그:** detect-counter 참조 엘리먼트 누락

### P1 (구조 변경)
4. **섹션 순서 재배치 권고:**
   ```
   HERO → HOW IT WORKS → AI 에디터(다크 축소) → 수익 증거 + 후기 → 안전성 → PRICING → FORM → FAQ
   ```
5. **다크 섹션 670줄 → 300줄 이하 압축:** "..." 패딩 제거, 화살표 축소
6. **pricing에 대행사 비교 추가:** "대행사 월 50만원 vs 월 6.5만원"

### P2 (콘텐츠 보강)
7. **업종별 사례 추가:** 뷰티/보험/요식업 before-after 최소 3개
8. **안전성 보장 pricing 근처 배치:** "계정 제재 0건" 배지
9. **윤윤숙용 "1:1 안내" 명시:** form 또는 pricing에 전화/카톡 안내
10. **김동혁용 ROI 계산:** "월 1건 문의만 늘어도 → 월 매출 XX만원"

### P3 (전환 최적화)
11. **무료체험 자동화:** 카카오톡 수동 → 즉시 이메일 발송
12. **쿠폰 "정가 87,000원" 근거 명시**
13. **FAQ를 form 이전으로 이동**

## 9. 한 줄 결론

**현재 페이지는 AI 에디터라는 강력한 무기를 갖고 있지만, 이것을 가격표 뒤에 숨겨놓는 구조적 실수로 인해 가장 설득력 있는 콘텐츠를 대부분의 방문자가 보지 못하고 이탈하고 있다.**

섹션 순서 재배치 + 다크 섹션 압축 + 무료체험 기간 불일치 수정이 가장 시급하며, 이 세 가지만으로도 전환율이 유의미하게 개선될 것으로 판단된다.
