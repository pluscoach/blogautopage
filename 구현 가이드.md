### 기본 구조

### SECTION 1: 히어로

**헤드라인:**
```
상위 1%
블로거들의 비밀 무기
AI 자동화 소통 솔루션
```
- "비밀 무기" → `text-naver` 초록 강조
- `text-4xl~[3.4rem] font-extrabold`

**좌측:**
- 태그라인 뱃지: "네이버 블로그 서이추 자동화"
- 수치 뱃지 3개: 하루 100명 / 수락률 87% / 990원
- CTA: "0원으로 24시간 무료 체험하기"
- 스크롤 힌트: "↓ 어떻게 작동하는지 확인하기"

**모바일 레이아웃:**
- 태그라인, 헤드라인, 뱃지, 스크롤 힌트 모두 **가운데 정렬**
- 수치 뱃지 3개: **세로 스택** (PC는 가로)
- `items-center md:items-start text-center md:text-left`

**우측 AI 데모:**
- 맥 스타일 윈도우 (빨/노/초 dots)
- 포스팅 분석 → 태그 추출 → AI 댓글 타이핑 애니메이션 (3개 댓글 순환)
- 소통 진행률 카운터
- float 애니메이션

### SECTION 2: 문제 공감 (STEP 1/2/3)

**공통:** `.pain-card` 스크롤 fade-in, 상단 다크 바, 세로 연결선

#### STEP 1: 서이추 무반응
- 헤드라인: "서이추 100명 보냈는데, (모바일 줄바꿈) 돌아온 건... **광고성 댓글**"
- 발송 완료 → 점 3개 기다림 UI → **빨간 배경** 광고성 답장 카드
  - `bg-red-50 border-2 border-red-200`
  - "광고성 복붙" 뱃지: `bg-red-500 text-white text-sm rounded-full`

#### STEP 2: 방문자 급락
- 헤드라인: "노가다로 올린 방문자... (모바일 줄바꿈) **하루 쉬니까 곤두박질**"
- **블로그 통계 스타일 그래프**:
  - 헤더: "일간 방문자 통계 / 최근 14일"
  - Y축 라벨: 500, 250, 0
  - **polyline으로 요동치며 상승 → 급락** (실제 블로그 통계처럼)
  - 초록선: 꾸준히 소통한 기간 (상승)
  - 빨간선: 하루 쉰 후 급락
  - 최고점 툴팁: "427명", 하락 끝: "34명"
  - X축: "1일 / 하루 쉼 / 14일"
- 하단 메시지: "꾸준히 소통해서 올린 방문자, **하루 쉬니까 뚝.**"

#### STEP 3: 포기
- 헤드라인: "블로그 글 쓰기보다 (모바일 줄바꿈) 이웃 늘리다가 **포기..**" (포기 빨간색)
- 네이버 에디터 UI + 1.5초 후 알림 카드 3개 덮어씀

### SECTION 3: 해결책 제시

**배경:** `bg-naverLight` (#E8F5E9)

**헤드라인:** ✨ "서이추부터 정성 댓글까지, 블로그 소통의 **'완전 자동화'**가 시작됩니다."

**4단계 순환 그리드 (2x2):**
- 중앙 AI 초록 원 + 점선 원형 보더
- 각 카드: `rounded-3xl shadow-md p-6~7`
  - STEP 1 타겟 분석: 적합 ✓ / 제외 ✕ (프로필+활동정보 포함)
  - STEP 2 서이추 신청: 발송 완료 리스트 + "외 48명 자동 발송 중.."
  - STEP 3 정성 댓글: AI 댓글 말풍선 (자연스러운 문체 + `..`)
  - STEP 4 이웃 관리: 블랙리스트(빨강)/자동 취소(노랑) pill 뱃지
- 제목: `text-2xl font-black`, 구분선으로 영역 분리

**CTA 버튼:** 흰 배경 + 초록→민트 그라데이션 텍스트, hover시 노란색
```css
.cta-trial:hover { background:#FACC15; }
.cta-trial:hover .cta-trial-text { -webkit-text-fill-color:#1A1A1A; }
```

**6단계 더보기 (`<details>`):**
- 다크 카드 (`bg-gray-900`) + AI 초록 아이콘 + "(클릭)" 뱃지
- 펼치면 다크 배경(`#0A0A0A`) 6단계 좌우 교차 레이아웃
- STEP 번호: `text-naver`, 설명: `text-gray-400` 기본 + 핵심 키워드 `text-yellow-300`
  - 01: "댓글창을 추적해 '진짜 활동 유저'"
  - 02: "서이추 차단 계정은 자동으로 걸러냅니다."
  - 03: "'알짜배기 계정'"
  - 04: "최신 포스팅을 실제로 읽고"
  - 05: "내 글을 진짜 읽었네?"
  - 06: "안전하게 퇴근합니다."
